03 — Units

Responsive Units

Cards sized using %, rem, and vw to demonstrate how each unit behaves.

% — Relative to parent
40%
60%

Percentage widths are always calculated relative to the container element.

rem — Relative to root font size
10rem
5rem

rem units stay consistent regardless of the parent's font size, making them great for spacing.

vw — Relative to viewport width
30vw

vw units scale directly with the browser window, useful for truly fluid elements.

% width

Percentage

Minimum column width set as a percentage of the container.

minmax(30%, 1fr)
rem width

Root Em

Fixed minimum using rem, which doesn't cascade from parents.

minmax(18rem, 1fr)
vw width

Viewport Width

Minimum tied to the viewport: always proportional to screen size.

minmax(20vw, 1fr)
px width

Pixels

Absolute minimum, never changes regardless of screen or font size.

minmax(240px, 1fr)
