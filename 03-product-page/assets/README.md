# 03 - Product Page 🛍️

A basic product page built with semantic HTML.

This project was focused on practicing foundational HTML elements used in product pages, such as images, links, and description lists. Even though it is a simple project, the goal was to write cleaner and more meaningful markup instead of only using generic `div` elements.

## Preview

![Product Page Preview](./assets/preview.png)


## Features

- Product image with descriptive `alt` text
- Product information section
- Navigation links using `<a>`
- Product details using description lists
- Semantic HTML structure
- Simple and readable layout

## Technologies Used

- HTML5
- CSS3

## What I Practiced

In this project, I practiced basic semantic HTML elements commonly used in product pages.

I used the `<img>` element with meaningful `alt` text to improve accessibility and provide a useful text alternative if the image cannot be displayed.

I also practiced using anchor tags with `<a>` to create navigation or external links.

For product specifications, I used description list elements:

```html
<dl>
  <dt>Material</dt>
  <dd>Cotton</dd>

  <dt>Size</dt>
  <dd>Medium</dd>
</dl>
