## üìñ Official Documentation

### ü•á Primary References

#### MDN Web Docs (Mozilla Developer Network)
**URL:** https://developer.mozilla.org/  
**Best For:** Everything web-related (HTML, CSS, JavaScript)  
**Why Essential:** Most accurate, most complete, constantly updated

**Key Sections:**
- [HTML Reference](https://developer.mozilla.org/en-US/docs/Web/HTML) - Every HTML element explained
- [CSS Reference](https://developer.mozilla.org/en-US/docs/Web/CSS) - All CSS properties and values
- [JavaScript Guide](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Guide) - Step-by-step JS learning
- [Web APIs](https://developer.mozilla.org/en-US/docs/Web/API) - DOM, Fetch, Storage, etc.

**How to Use MDN:**
- Search: "mdn [topic]" in Google
- Read the description + examples
- Check browser compatibility tables
- Try the interactive examples

---

#### JavaScript.info
**URL:** https://javascript.info/  
**Best For:** Modern JavaScript from basics to advanced  
**Why Recommended:** Clear explanations, interactive exercises, comprehensive

**Key Sections:**
- Part 1: JavaScript Fundamentals (variables, functions, objects)
- Part 2: Browser APIs (DOM, Events, Forms)
- Part 3: Advanced Topics (Async, Modules, Network requests)

**Recommended Path:**
1. Read chapters sequentially
2. Complete the exercises at the end
3. Experiment in browser console
4. Apply to your projects
